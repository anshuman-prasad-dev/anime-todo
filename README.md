# Anime Todo ✨

A beautiful, lightweight, and floating anime-styled desktop todo list widget built with Electron. Designed with a stunning pastel UI, smooth animations, and sakura particle effects to keep you productive and relaxed.

## Features

- **Floating Desktop Widget**: Stays precisely where you need it with transparency and smooth window controls.
- **Anime-Inspired Aesthetics**: Features a custom sakura (cherry blossom) canvas particle system, pastel gradients, and glassmorphism.
- **Real-Time Progress Tracking**: Circular SVG progress ring dynamically updates as you complete tasks.
- **Task Filtering**: Easily switch between All, Active, and Done quests.
- **Tray Integration**: minimize to system tray to keep your desktop clean.
- **Local Storage**: Automatically saves your tasks locally.

## Tech Stack

- **Electron**: Desktop application framework.
- **HTML5 & CSS3**: Custom animations, transitions, and SVG manipulation.
- **Vanilla JavaScript**: Lightweight state management and DOM manipulation without external libraries.

## Installation

1. Clone or download this repository.
2. Ensure you have Node.js installed.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```
